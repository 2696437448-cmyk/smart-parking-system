# 论文材料目录说明

本目录用于存放 `smart-parking-thesis` 的本科毕业论文配套材料草稿，整理原则是“学校模板优先、项目事实约束、正文可直接二次套版”。当前内容已经按 `reference/` 中的学校模板继续收敛，其中 5 份属于一对一正文材料，另有 1 份答辩辅助稿和 1 份提示词文档。

## 一对一模板正文

1. `task-book.md`
   - 对应 `reference/任务书_*.docx`。
   - 已按任务书的栏目顺序整理为：基本信息、题目、主要内容、基本要求、原始资料与参考文献、进度计划、院系意见。

2. `proposal.md`
   - 对应 `reference/开题报告_*.docx`。
   - 已按开题报告栏目整理为：课题名称与来源、研究意义、研究现状、研究内容、研究思路与方法、主要参考文献、进度安排、教师意见、所在系意见。

3. `midterm-review.md`
   - 对应 `reference/中期检查_*.docx`。
   - 已按学校中期检查表整理为：进度计划、目前已经完成内容、尚需完成内容、存在的问题和拟采取的办法。
   - 文风按“项目时间过半”的中期状态编写，没有直接写成最终完成态。

4. `process-record.md`
   - 对应 `reference/指导过程记录表_*.doc`。
   - 已按阶段记录方式整理 15 次指导内容，后续可直接补日期、签字与纸质归档字段。
   - 当前版本已补入统一登录、角色隔离和网关鉴权收口等项目后期完善内容。

5. `thesis-draft.md`
   - 对应 `reference/本科毕业设计说明书模板.doc`。
   - 已按说明书常见结构整理为：题目页、中文摘要、英文摘要、目录、正文各章、结论与展望、参考文献、致谢。
   - 当前版本偏“说明书正文初稿”，适合继续套入学校 Word 模板细修。

## 辅助材料

1. `defense-script.md`
   - 最终答辩口播稿。
   - 内容与说明书、任务书、开题报告保持一致，但表达更接近学生现场陈述。

2. `writing-prompts.md`
   - 论文生成与润色提示词文档。
   - Prompt 本体统一使用英文编写，实际生成目标固定为中文论文材料。

## 内容依据

本目录正文内容主要依据以下项目材料整理：

1. 项目总览与默认完成态：`README.md`
2. 系统架构与业务链路：`memory-bank/architecture.md`
3. 论文证据索引：`reports/thesis_evidence_package.md`
4. 综合验收结论：`reports/step40_technical_acceptance.md`
5. 登录与认证补充材料：`docs/superpowers/specs/2026-04-21-smart-parking-login-design.md`、`reports/step43_login_auth_execution.md`
6. 学校表单口径参考：`reference/任务书_*.docx`、`reference/开题报告_*.docx`、`reference/中期检查_*.docx`、`reference/指导过程记录表_*.doc`、`reference/本科毕业设计说明书模板.doc`
7. 论文生成提示词参考：`writing-prompts.md`

## 使用建议

1. 先看 `writing-prompts.md`，选择要生成或修改的英文 Prompt。
2. 再看本目录已经整理好的 5 份模板正文，优先沿用现有栏目结构。
3. 需要答辩口播时，直接在 `defense-script.md` 基础上继续按个人习惯压语言。
4. 最终提交前，将正文复制到学校 Word 模板中，补齐封面、签字栏、日期栏和格式细节。
5. 定稿前，再用 `writing-prompts.md` 中的自检 Prompt 检查栏目一致性、项目事实一致性和语言自然度。
