# Labs：配套练习册

`labs/` 的作用不是考试，而是防止你停在“看过但没学会”。

使用方式：

1. 先读对应 `chain`
2. 再做对应 `lab`
3. 不要先翻答案，先自己答
4. 最后对照“为什么答案对 / 常见错因”修正理解

对应关系：

- `lab-01-owner-dashboard.md` 对应 `chain-01`
- `lab-02-owner-business.md` 对应 `chain-02`
- `lab-03-admin-realtime.md` 对应 `chain-03`
- `lab-04-model-etl.md` 对应 `chain-04`
