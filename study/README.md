# Smart Parking 学习包

这套资料不再只做“读代码导航”，而是把学习拆成 3 层：

- `weeks/`：看地图，知道项目有哪些模块、当前主线是什么
- `chains/`：跟主链，手把手把一条功能链读懂
- `labs/`：做练习，用题目、验证和复述检查自己是不是真的会了

如果你是代码基础较弱、第一次读这个项目，默认不要直接从 `weeks/` 开始一路顺读。更推荐下面这条主线：

1. [guide/start-here.md](./guide/start-here.md)
2. [chains/README.md](./chains/README.md)
3. 先学 `chain-01`
4. 做完对应 `lab-01`
5. 再进入下一条链

## 三种入口

### 1. 我只想先建立系统地图

走这里：

- [weeks/](./weeks/)
- [tracks/4-week-intensive.md](./tracks/4-week-intensive.md)

适合：

- 第一次见到这个项目，先想知道它由哪些服务组成
- 想快速知道 Step40 当前主线是什么
- 还没准备深入代码细节

### 2. 我想真正把代码主链学懂

走这里：

- [guide/start-here.md](./guide/start-here.md)
- [chains/README.md](./chains/README.md)

适合：

- 看“周计划”会知道学什么，但还是不会真的跟代码
- 希望有人明确告诉你：先看哪里、看到什么、这一步该得出什么结论
- 想从页面、接口、模型脚本一路跟到真实代码

### 3. 我想检查自己是不是真的学会了

走这里：

- [labs/README.md](./labs/README.md)
- [templates/chain-note-template.md](./templates/chain-note-template.md)
- [templates/lab-answer-template.md](./templates/lab-answer-template.md)

适合：

- 不想停在“我好像看过”
- 想通过定位题、字段追踪题和最小验证确认自己会了
- 想留下答辩式表达材料
