# Chains：按功能链学习代码

`chains/` 是这套学习体系真正的主教材。它和 `weeks/` 最大的区别不是“更长”，而是“真的带你跟代码”。

每一条链都包含：

- 这条链在业务上解决什么问题
- 场景故事
- 分步跟读路线
- 每一步要回答的问题
- 每一步的参考答案
- 理解题、定位题、字段追踪题
- 最小验证
- 1 分钟复述稿

## 推荐顺序

1. [chain-01-owner-dashboard-request.md](./chain-01-owner-dashboard-request.md)
2. [chain-02-owner-order-billing-navigation.md](./chain-02-owner-order-billing-navigation.md)
3. [chain-03-admin-dashboard-realtime.md](./chain-03-admin-dashboard-realtime.md)
4. [chain-04-etl-model-registry-acceptance.md](./chain-04-etl-model-registry-acceptance.md)

## 学习方法

每读完一条链，都立刻去做对应的 lab：

- `chain-01` 对应 `lab-01`
- `chain-02` 对应 `lab-02`
- `chain-03` 对应 `lab-03`
- `chain-04` 对应 `lab-04`

不建议连续读两条链却不做题，否则很容易又回到“看过但没学会”的状态。
