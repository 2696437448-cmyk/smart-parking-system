# PRD - 智慧停车调度与共享系统

## 1. 项目目标

构建一个可部署、可答辩复现的社区智慧停车演示系统，实现：

1. 短时车位供需缺口预测。
2. 动态车位分配调度。
3. 业主预约与计费流程。
4. 物业监控与收益统计展示。

## 2. 必须实现功能

1. 多源数据融合（车位状态、车辆进出、业主行为画像）。
2. 使用 Spark 完成数据清洗与特征对齐。
3. 基于轻量 LSTM 的时序预测。
4. 基于匈牙利算法的动态调度。
5. 业主侧界面：预约、预估价格、导航提示。
6. 物业侧界面：占用监控、调度触发、收益看板。
7. 可靠性基线：幂等、降级、DLQ、可观测性看板。

## 3. 非目标（本期不做）

1. 原生移动端上架。
2. 城市级高并发实时流式生产部署。
3. 多地域容灾与跨机房高可用。
4. 完整 MLOps 平台（仅保留轻量模型注册与激活）。

## 4. 相关角色

1. 学生开发者（实现、实验、论文）。
2. 指导老师与评审（验收与答辩）。
3. 模拟业主用户（预约流程演示）。
4. 模拟物业运营者（监控与调度演示）。

## 5. 验收标准

1. 单机 Docker 可完整跑通端到端流程。
2. 并发预约无超卖、无重复扣费。
3. 模型服务故障时系统可降级，不出现硬失败。
4. MQ 失败可重试并进入 DLQ，过程可追踪。
5. WebSocket 断开后自动切轮询，页面持续更新。
6. README 提供可复现实验命令与预期结果。
